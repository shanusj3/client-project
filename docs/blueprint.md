# **App Name**: Sampat Insights

## Core Features:

- Professional Profile Display: Present detailed sections covering Executive Summary, Core Competencies, Professional Journey (timeline), Specialized Credentials, and Education using the provided content.
- Interactive Contact System: An interactive section with contact details (email, LinkedIn) and a form to facilitate inquiries from potential clients and collaborators.
- Dynamic Navigation Bar: A responsive, sticky navigation menu that allows seamless browsing and quick access to different sections of the profile.
- AI-powered Expertise Inquirer: A conversational AI tool that can respond to visitor questions about Rajesh Sampat's professional background, expertise, and services by analyzing the site's content.
- Mobile-First Responsiveness: Ensure the website is fully responsive and offers an optimal viewing and interaction experience across various devices and screen sizes.

## Style Guidelines:

- Primary Color: A vibrant chartreuse (#BAEB17) used for key interactive elements, headings, and branding accents to create a striking contrast against the dark background, conveying modernity and energy.
- Background Color: A deep, dark charcoal (#171F14) with a subtle greenish tint, providing a sophisticated, immersive, and minimalist canvas for the content.
- Accent Color: #1A2A6C employed sparingly for critical call-to-action buttons, highlights, and secondary interactive states to grab attention.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, bold, and impactful presence. Body text font: 'Inter' (sans-serif) for clear readability and a professional, objective tone.
- Use a consistent set of clean, minimalist line icons that align with the professional and modern aesthetic, ensuring easy comprehension and visual elegance.
- Employ a modular layout with clear separation between content blocks, generous darkspace (negative space), and a strong visual hierarchy to guide user attention. Adopt full-width hero sections tapering into centered, contained content areas, similar to the provided design reference.
- Incorporate subtle, tasteful animations for transitions between sections, hover effects on interactive components, and state changes to enhance user engagement without being distracting.
